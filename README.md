

# Mathing Gem POC

The main objective is provide a easy way to compare skills, preferences or any common criteria between 2 models.  


## Versions
	Ruby 2.3.3
	Rails 4.2.7.1

## Steps to build

	```bundle install```

	```rake db:create```

	```rake db:migrate```

	```rake db:seeds```

	```bundle exec puma```
